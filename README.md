# Web-Based URL Detection System
This project is a web-based application designed to analyze URls and identify potentially malicious or phishing links using external threat intelligence APIs.

## Features
- User-freiendly web interface for URL input
- URl Validation and analysis
- Risk indication based on Api response

## Purpose
This project was developed as part of an academic curriculum to understand how web applications can integratte security APIs for real-world threat detection.

## Tech Stack
- Python
- WEb framework (Flask/Django)
- HTML, CSS
